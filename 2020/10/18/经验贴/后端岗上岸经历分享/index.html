    普通本科生怎么准备

    0.小插曲：

    • 我是大二下学期的时候准备下一年的秋招，自学Java语言也是大二下学期开始的。谈起来我为什么学习Java，学习后端，中间还有一个小插曲：大二上学期学校开了Java语言程序设计这门课，但是我当时比较贪玩，所以就没好好学习，所以在期末考试的时候，我的Java考试，我一度认为我肯定是挂了的、我后面两道大题是瞎写的，前面选择填空也好多不会，我当时连static关键字的作用都不知道是啥。所以在大二寒假，我就在家里复习补考Java，又因为当时我的教务部密码忘记了，查不了成绩，我就乖乖地耐心等待开学辅导员给我发补考通知。但是开了学我才知道我Java课程没挂，老师高抬贵手，给了我一个及格分。沃日，我瞬间感觉我整个人都不好了，我放弃了寒假美美的玩耍的机会在家里学习，真的太“脑残”了。但是正是因为我寒假学习了，我才发现Java也没有那么难，之后就有一直在学习。所以阴差阳错，让我开始了我的后端学习之路。也是从这个时候开始我决定我要直接工作，不会考研。

    1. 大二下：

    • 大二下其实还是一直在看Java，顺便学习了前端相关的知识。我记得我当时看的Javase 基础是的B站的siki学院的课程。
      Java编程学习第一季
      Java编程学习第二季
      Java编程学习第三季
      Java编程学习第四季
    • siki学院的Java基础课还是不错的，但是寒假就跟着这个准备我的“补考”的。

    2. 大二暑假：

    • 大二暑假在家就是边学车，边学习，当时主要看的是数据结构与算法，大二开了这门课，但是也是没有好好听，期末抱佛脚的水平罢了。当时看的是那本《啊哈，算法》那本书，当时买的时候不知道里面是c++，但里面的算法原理还是描述的很生动形象的。同时当时还看了《算法》黄皮书，那本书，怎么说呢，Java写的，但是比较难啃，而且他的输入输出不是Java标准的输入输出。最后也是看了一半，没有坚持看完。

    3. 大三九月份：

    • 大三开始，学校开了计算机网络，操作系统，这两门课，这个时候我已经加入了牛客，知道了面试的重点也是这块，所以这两门课基本就是很认真的听课，然后记笔记，也为之后实习秋招复习提供了很好的参考资料。同时也看了cyc-2018的关于计网操作系统的笔记，总结相关的知识点。
      cyc2018知识点
    • 同时9月份，自己开始学习Java web，主要就是Servlet，jsp。主要就是request，response这两个对象的方法是重点。至于jsp部分，网上有很多人说jsp已经过时了，我认为自己简单了解一下就好，不用过于深究。
      当时看的是狂神说的视频。
      狂神说 Java Web教程

    4. 大三十月份十一月份：

    • 主要学习的就是SSM框架，因为当时我不理解Spring的AOP IOC，所以当时我花了挺长时间在这上面，也买了一些书来看，当时Spring 这部分我看了四五遍才理解。主要看的也是狂神说的SSM教程，看完SSM，可以自己找一个SSM的项目来做一下。
      狂神说 Spring教程
      狂神说 Spring MVC教程
      狂神说 Mybatis教程

    5. 大三十二月份：

    • 进入十二月份，这个阶段我开始复习前面学习的Java se中的重点知识，比如集合，多线程，并发，反射这些知识点。同时配合一些专栏来看，比如极客时间上面的专栏，同时买来了《Java编程思想》《Java核心技术》《Java多线程编程艺术》《Java并发编程艺术》这些书来看。因为有了前面Java se的基础，所以我看这些书都比较得心应手，看起来都比较快。并且配合牛客的面经，总结一些常见的面试题。

    6.大三一月份：

    • 花了几天时间看JVM，视频配合书籍，视频看的是狂神说的JVM，书就是那本经典的《深入理解JVM》，我看这本书的时候，没有按照目录的要求来看，因为我觉得这本书确实写的很理论化，所以重点看几章。重点看：第二章，第三章，第七章，第八章。这4章对待面试应该是没有问题了。其他的章节我没有看。
    • 其余时间专门学习了Mysql，也是视频配合书籍，书籍就是那本《高性能Mysql》，同时还看了极客时间的Mysql专栏。但是书的话我也是挑重点看。专栏也是，最后配合牛客的面经，总结一些常见的Mysql的面试题。

    7. 大三二月份：

    • 这个时候主要学的就是Spring Boot，视频是看的尚硅谷的Spring Boot视频，Boot学完之后，然后学长推荐了牛客网的高级项目课的课程，学长直接把他的给了我，高级项目课就是牛客网讨论区的简单实现。用的技术就是Spring Boot + Mysql + Redis + kafka + ElasticSearch这些。自己花了15天时间，成功做出来部署成功。
    • 因为自己没有学习过Redis，kafka，es这些，所以在学习完这个项目之后，我就自己重点学习Redis，Kakfa等技术。书看的是《Redis深度历险》《Redis实战》。我觉得有些技术点 我们还是先学会在项目中运用，然后反过头来再去深入理解。这种学习方法还是比较好的。
    • 之后自己学习了微服务Spring cloud相关的知识，视频看的也是尚硅谷的Spring Cloud视频。

    尚硅谷Spring Cloud视频
    尚硅谷Spring Boot视频

    8. 疫情在家：

    • 整个疫情在家，每天的学习就是早上10点起床：背自己总结的面试题，下午的话刷Leetcode，总结。晚上看左程云大佬的算法课程。算法题一共是刷了150+，刷的不多，主要还是因为自己菜，有的题不看答案就是不会做。最后就是准备实习简历，参加春招实习，之后参加秋招的经历了。秋招的面经可以看我发的帖子。

    • 然后疫情在家，就是完善项目了，之前做了一个牛客网的高级项目课了，所以我对那个基本的流程已经很熟悉了，但是我想对他的内容做了一个扩展，于是参考了尼特社区，这个Java写的社区项目，这个项目前端是有完整的模板的，所以相当于我们自己需要写后端实现，它的前端是lay ui里面的一个社区模板，后端就是自己写，按照他前端的功能自己实现，完成这个项目我大概用了两个月的时间。最后我也是部署上线，这个项目面试的时候也帮助了我很大的忙，基本问项目上的问题我都能够回答出来。

      五、总结

    • 上面是我准备秋招的总的时间线，所以准备整个秋招，我基本花了一年时间，从大二下到大三下，跨度大，但自己还是坚持了下来。希望对来年参加秋招的学弟学妹有参考价值，如果自己不是那种3个月逆袭的大佬，我们还是提前准备，稳扎稳打，一步一个脚印。

    • 对于普通本科生来说，如果自己不是那种大佬级别人物，咱们的策略就是海投，如果别人相中了咱，咱们就好好准备面试。

    • 我很庆幸我自己在牛客上找了一个模拟面试的小伙伴，我俩准备实习，秋招的时候都会**模拟面试**，一聊就是好几个小时的那种，大家也可以找一个，利用好牛爱网这个平台。

    • 普通的本科生不要灰心，金字塔尖的终究是少数人，大部分人还都是普通人，摆正心态，我们要想办法在普通人里面脱颖而出。

    • 项目部署成功，一定要放上项目链接，你给面试官描述3分钟，不如面试官点进去你的链接看3秒钟。面试官点链接进去之后，基本对你的工程量有一个大致的预估，然后他才会针对你项目上的细节进行提问。起码面试官知道这个项目是真实的，不是你凭空捏造背下来的。

    • 能面试就面试，我们就是在不断的面试中进步的，我们永远没有准备充分的那一天。

    • 一定要尽早投递简历，金九银十已经不复存在了，好的公司是金七银八，很有可能到时候没有hc了

    • 抓住提前批，一般都免笔试，笔试要比面试时候手撕代码难。

    • 简历不要造假，你的简历就是面试官的题库，你的简历相当于是在给自己出题，不要给自己出难题，但也有小概率超纲题

    • 自己定好目标，不要放弃，干就完了，奥利给
