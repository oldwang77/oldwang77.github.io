秋招经验6

我个人推荐的学习路线：

Java

Java 基础：b站上非常多资源，黑马和尚硅谷的都可，书的话我觉得《Java核心卷》和《Java程序设计语言》都可。

Java 并发：我看了《并发编程的艺术》这本书，还不错的，重点就是 volatile、synchronized、JMM、各种锁、并发工具包、CAS。

Java 虚拟机：感觉理论偏多，我主要看了《深入理解JVM》，感觉面试重点就是运行时数据区和类加载机制、GC。

Spring 框架：黑马和尚硅谷的视频都不错，不过我是看了”楠哥教你学Java“的SSM，感觉更加简洁易懂。

Java 的话，个人推荐先动手，多看视频多操作，要面试深入理解知识点再看书。前期主要以视频为主，多动手，后期以自己的总结和书为主，多背题。像我整理基础知识点的时候，都是同时看五六本书+CSDN+牛客来总结的，特别麻烦。

数据库

MySQL：这个应该是基础必学的，我主要看的是尚硅谷李玉婷老师的MySQL视频，b站也可以搜到，感觉很不错。书的话看的就是《高性能MySQL》，重点我只看了前几章，就是锁、存储引擎、索引、优化这些。

Redis：这个对于想去好一点互联网的同学来说应该也是必需的，b站黑马程序员的redis课程蛮不错的，书的话《Redis开发与运维》我觉得非常不错，知识点很清晰。

计算机基础

计算机网络：这个算是我最拿得出手的部分了，硬背就完事了..主要看了谢希仁的《计算机网络第七版》，感觉足够了，重点就是TCP和HTTP了。

操作系统：我主要看的就是王道的操作系统，感觉对我来说算足够了，重点就是进程、线程、死锁。有的公司可能会问页面调度算法，FIFO、LRU 啥的。

数据结构：感觉直接刷算法就可，leetcode 搞起来，我一开始第一题都不会做，熟能生巧吧，虽然现在也菜，但比当时好多了，这个就是靠天赋，没有天赋就是积少成多了。

计算机组成原理：没看过。

面试的话，说实话主要就是靠背，毕竟运气啥的咱不能决定，背的话我还有个绝技，就是自己录音，然后睡觉的时候一直听，当时考研靠这个一周背过了肖四2万字的大题。。技术的话大概就这么多了，之后都是我的无用吐槽，可以不看哈哈哈哈。

心态
心态是个很重要的东西，怎么说呢，活着不就是为了生活好吗，生活好不就是为了开心些吗。。当我看到这种话的时候我感觉这就是废话，确实，别人也对我说过类似的话，我心里想，我找不到工作，我心情踏马能好吗？淦。

刚来牛客的时候，我觉得这里都是神一样的人物，这里就是个学习圣地，我每天不刷几个小时都难受，我从来没想过我这样的废物也能混到个红名。。。还有1000个粉丝我真的超级惭愧。。以后我也没啥技术贡献了，所以可以安心取关我了。。我可是本科考研前i++和++i不分的人啊，话说之前压根就没想做这一行，在虎扑发帖问计算机专业不会编程能做什么被人疯狂嘲讽，我考研还是专业第一，但我确实不会编程，所以决定走这行也下了蛮大决心（好像也没啥决心，我其它也不会啥）。

在去年的8月27日，我参加了某不知名小厂（一个月工资1千）的招聘，一问三不知，面向对象的特性，不知道。。说几个你知道的排序？堆排序（其它全忘了，堆排序还是想了很久）。。你这是学计算机的吗？我说我没好好学。。真尴尬得要命，当场回家，下楼就买了两本书，C#的，后来又卖了- -

在同学的”教唆”下选择了Java，现在来看也不后悔，对于基础薄弱的人来说，Java确实是不错的选择，很容易上手。本科入学不是C语言和C++我真觉得我能再抢救一下的，淦腻🐎的指针。。。

一开始我跟所有人一样焦虑，大概是从9月15日开始学Java的..由于自己相当于半个零基础，真的很慌，我连个publict int add(int a,int b) return a + b;这种方法都不会写。。当初的梦想就是一年10万，10万我都觉得多了，我感觉自己就是个废物啥都不会，凭啥给我发10万工资哈哈哈哈。然后反正就一听学长去中兴，一听谁去浦发，一听20万一听钱就脑子很热。。感觉脑子里就只有工资，觉得自己一定要赚20万啥的。

每天就狂学，脑子疼，是真的每天脑子疼，太焦虑了，不停地掉头发。。跟同学去网吧，别人打游戏我刷牛客选择题；跟同学吃烧烤，别人吃东西，我刷牛客选择题；跟同学去酒吧，别人聊天，我刷牛客选择题。。好像做这些能换来一点心安似的。。9-12月大概都是这个心态吧。。

1月的时候，疫情来了，我这人很怕死，当时就觉得还是命最重要，家里还来了武汉的，每天吓得也没心情学习了，好多天一天就光做一道算法题。。然后我就一直想着去个银行得了，也没咋学，这算是第二个阶段吧，1-3月都这样。

第三个阶段是4-6月，一开始没准备投简历的，因为就想去银行国企了，也没啥实习可参加。后来boss直聘上的阿里和华为疯狂轰炸我，我就参加了阿里。。第一次笔试面试就是阿里的，笔试0分还面我了，还进了2面，可能我因此膨胀了，就开始疯狂投简历，开始了自己的春招实习。。反正后面一共投了20多家，就过了2个，哈哈。这段时间边面试边学习，感觉也是自己最强的时候了。

最后一个阶段，7月到现在，没咋学习了，每天就看看自己的总结，不断地缩减，添加，也没背，反正天天看来看去多少也有记忆了。直到今天签约了农行，开始还是蛮纠结的，后来觉得也就罢了，自己学那么多为了钱为什么不去互联网呢，不就是为了轻松点吗。。看别人都签了，下午最后我也就签了，其实自己不缺钱，有房，但都是家里给的，家里啥也不剩，开始想去浙商也是想多给家里点钱。

反正我觉得吧，弄清楚自己想要什么真的是非常重要的一件事情，不要像我一开始一样，一看别人都去互联网赚大钱，一看见钱，一听见几十万什么的就迷失了自己，就跟着瞎搞，就跟着瞎焦虑。。无论是去哪里，定好自己的目标，按部就班，朝着那个目标不断努力，结果是不会让你后悔的。

最后想说什么，就是真的不要做无畏的焦虑，付出多少就会收获多少，烦恼无用，劳逸结合，弄清楚自己想要什么，然后朝着这个方向努力就是了，不要受外界过多影响了。

最后再说一句，湖人总冠军。

最后再说一句，祝2021届都拿到自己想要的offer，2022届及之后明年秋招都顺利。

最后再说一句，农行永远的神，明年请不要挂我转正再给我个S+绩效吧哈哈哈！

普通本科生怎么准备

0.小插曲：

  • 我是大二下学期的时候准备下一年的秋招，自学Java语言也是大二下学期开始的。谈起来我为什么学习Java，学习后端，中间还有一个小插曲：大二上学期学校开了Java语言程序设计这门课，但是我当时比较贪玩，所以就没好好学习，所以在期末考试的时候，我的Java考试，我一度认为我肯定是挂了的、我后面两道大题是瞎写的，前面选择填空也好多不会，我当时连static关键字的作用都不知道是啥。所以在大二寒假，我就在家里复习补考Java，又因为当时我的教务部密码忘记了，查不了成绩，我就乖乖地耐心等待开学辅导员给我发补考通知。但是开了学我才知道我Java课程没挂，老师高抬贵手，给了我一个及格分。沃日，我瞬间感觉我整个人都不好了，我放弃了寒假美美的玩耍的机会在家里学习，真的太“脑残”了。但是正是因为我寒假学习了，我才发现Java也没有那么难，之后就有一直在学习。所以阴差阳错，让我开始了我的后端学习之路。也是从这个时候开始我决定我要直接工作，不会考研。

1. 大二下：

  • 大二下其实还是一直在看Java，顺便学习了前端相关的知识。我记得我当时看的Javase 基础是的B站的siki学院的课程。
    Java编程学习第一季

  • siki学院的Java基础课还是不错的，但是寒假就跟着这个准备我的“补考”的。

2. 大二暑假：

  • 大二暑假在家就是边学车，边学习，当时主要看的是数据结构与算法，大二开了这门课，但是也是没有好好听，期末抱佛脚的水平罢了。当时看的是那本《啊哈，算法》那本书，当时买的时候不知道里面是c++，但里面的算法原理还是描述的很生动形象的。同时当时还看了《算法》黄皮书，那本书，怎么说呢，Java写的，但是比较难啃，而且他的输入输出不是Java标准的输入输出。最后也是看了一半，没有坚持看完。

3. 大三九月份：

  • 大三开始，学校开了计算机网络，操作系统，这两门课，这个时候我已经加入了牛客，知道了面试的重点也是这块，所以这两门课基本就是很认真的听课，然后记笔记，也为之后实习秋招复习提供了很好的参考资料。同时也看了cyc-2018的关于计网操作系统的笔记，总结相关的知识点。
    cyc2018知识点
  • 同时9月份，自己开始学习Java web，主要就是Servlet，jsp。主要就是request，response这两个对象的方法是重点。至于jsp部分，网上有很多人说jsp已经过时了，我认为自己简单了解一下就好，不用过于深究。
    当时看的是狂神说的视频。
    狂神说 Java Web教程

4. 大三十月份十一月份：

  • 主要学习的就是SSM框架，因为当时我不理解Spring的AOP IOC，所以当时我花了挺长时间在这上面，也买了一些书来看，当时Spring 这部分我看了四五遍才理解。主要看的也是狂神说的SSM教程，看完SSM，可以自己找一个SSM的项目来做一下。
    狂神说 Spring教程
    狂神说 Spring MVC教程
    狂神说 Mybatis教程

5. 大三十二月份：

  • 进入十二月份，这个阶段我开始复习前面学习的Java se中的重点知识，比如集合，多线程，并发，反射这些知识点。同时配合一些专栏来看，比如极客时间上面的专栏，同时买来了《Java编程思想》《Java核心技术》《Java多线程编程艺术》《Java并发编程艺术》这些书来看。因为有了前面Java se的基础，所以我看这些书都比较得心应手，看起来都比较快。并且配合牛客的面经，总结一些常见的面试题。

6.大三一月份：

  • 花了几天时间看JVM，视频配合书籍，视频看的是狂神说的JVM，书就是那本经典的《深入理解JVM》，我看这本书的时候，没有按照目录的要求来看，因为我觉得这本书确实写的很理论化，所以重点看几章。重点看：第二章，第三章，第七章，第八章。这4章对待面试应该是没有问题了。其他的章节我没有看。
  • 其余时间专门学习了Mysql，也是视频配合书籍，书籍就是那本《高性能Mysql》，同时还看了极客时间的Mysql专栏。但是书的话我也是挑重点看。专栏也是，最后配合牛客的面经，总结一些常见的Mysql的面试题。

7. 大三二月份：

  • 这个时候主要学的就是Spring Boot，视频是看的尚硅谷的Spring Boot视频，Boot学完之后，然后学长推荐了牛客网的高级项目课的课程，学长直接把他的给了我，高级项目课就是牛客网讨论区的简单实现。用的技术就是Spring Boot + Mysql + Redis + kafka + ElasticSearch这些。自己花了15天时间，成功做出来部署成功。
  • 因为自己没有学习过Redis，kafka，es这些，所以在学习完这个项目之后，我就自己重点学习Redis，Kakfa等技术。书看的是《Redis深度历险》《Redis实战》。我觉得有些技术点 我们还是先学会在项目中运用，然后反过头来再去深入理解。这种学习方法还是比较好的。
  • 之后自己学习了微服务Spring cloud相关的知识，视频看的也是尚硅谷的Spring Cloud视频。

尚硅谷Spring Cloud视频
尚硅谷Spring Boot视频

8. 疫情在家：

  • 整个疫情在家，每天的学习就是早上10点起床：背自己总结的面试题，下午的话刷Leetcode，总结。晚上看左程云大佬的算法课程。算法题一共是刷了150+，刷的不多，主要还是因为自己菜，有的题不看答案就是不会做。最后就是准备实习简历，参加春招实习，之后参加秋招的经历了。秋招的面经可以看我发的帖子。

  • 然后疫情在家，就是完善项目了，之前做了一个牛客网的高级项目课了，所以我对那个基本的流程已经很熟悉了，但是我想对他的内容做了一个扩展，于是参考了尼特社区，这个Java写的社区项目，这个项目前端是有完整的模板的，所以相当于我们自己需要写后端实现，它的前端是lay ui里面的一个社区模板，后端就是自己写，按照他前端的功能自己实现，完成这个项目我大概用了两个月的时间。最后我也是部署上线，这个项目面试的时候也帮助了我很大的忙，基本问项目上的问题我都能够回答出来。

    五、总结

  • 上面是我准备秋招的总的时间线，所以准备整个秋招，我基本花了一年时间，从大二下到大三下，跨度大，但自己还是坚持了下来。希望对来年参加秋招的学弟学妹有参考价值，如果自己不是那种3个月逆袭的大佬，我们还是提前准备，稳扎稳打，一步一个脚印。

  • 对于普通本科生来说，如果自己不是那种大佬级别人物，咱们的策略就是海投，如果别人相中了咱，咱们就好好准备面试。

  • 我很庆幸我自己在牛客上找了一个模拟面试的小伙伴，我俩准备实习，秋招的时候都会**模拟面试**，一聊就是好几个小时的那种，大家也可以找一个，利用好牛爱网这个平台。

  • 普通的本科生不要灰心，金字塔尖的终究是少数人，大部分人还都是普通人，摆正心态，我们要想办法在普通人里面脱颖而出。

  • 项目部署成功，一定要放上项目链接，你给面试官描述3分钟，不如面试官点进去你的链接看3秒钟。面试官点链接进去之后，基本对你的工程量有一个大致的预估，然后他才会针对你项目上的细节进行提问。起码面试官知道这个项目是真实的，不是你凭空捏造背下来的。

  • 能面试就面试，我们就是在不断的面试中进步的，我们永远没有准备充分的那一天。

  • 一定要尽早投递简历，金九银十已经不复存在了，好的公司是金七银八，很有可能到时候没有hc了

  • 抓住提前批，一般都免笔试，笔试要比面试时候手撕代码难。

  • 简历不要造假，你的简历就是面试官的题库，你的简历相当于是在给自己出题，不要给自己出难题，但也有小概率超纲题

  • 自己定好目标，不要放弃，干就完了，奥利给

4.1 简历

推荐使用《超级简历》的学生求职模板，我个人感觉写出来非常精简，也很清爽。

4.3 面试技巧

我觉得我的技巧一般，属于比较常规的套路，从自我介绍开始说
我叫XXX，是XX大学的应届生，我做过两个项目，分别介绍自己做了什么，但是不要说的太长，拿重要的说，因为面试官大多不会记住你说了什么，还是主要会照着简历问。这之后我会重点强调我很会并发编程和JVM（你问我就行了，问我就上了我的套儿），最后几句做一个总结，其实我觉得自我介绍更多的还是面试的一个开场白，一定要非常流利。

另外我面试的时候属于主动交代型，这样有时候好，有时候不好，不好的时候就是会让面试官觉得你在引导他，他反而不会这么去问也不会太开心。什么叫主动交代型？咱比如说，面试官让我说说HashMap，我基本上就会一通说好多，而且说完我还会说我比较了解ConcurrentHashMap，就是把面试官围在我会的东西里与他切磋，有时效果很好，有时不太好，还得是大家自己考量。

另一方面就是自信，这个我没办法教大家，我自身相对来说是一个比较自信的人，但是请大家一定要表现出自己很可以的样子，这样也给面试官留一个好的印象。

校招知识科普

什么是应届生？

百度百科的解是：应届毕业生是指在应该毕业的年份毕业的毕业生。通俗来讲，一般情况下就是明年毕业的大学生。（ 海外学生和研究生有些不是这种情况）比如我是2021年7月毕业，那么现在我就是应届生（2021届），我参加的就是2020年的秋招。

什么是HC？

HC = head count，江湖人称 招聘人数。

什么是春招，秋招提前批，秋招，补录？

  • 春招：一般在3~5月份，面向应届生，主要是在秋招时，公司没有招够，进行少量的补招。
  • 秋招提前批：一般6~7月份下旬，中大厂对简历优秀（学历+含金量高的比赛+大厂实习）的选手进行选拔。
  • 秋招：一般在810月份，常说的金九银十，其实十月份的机会（名企）就比较少了，主要面试还是集中在8月下旬9月份。
  • 补录：一般在10~11月份，拿了很多个offer的同学只能选择一个，就会释放很多hc，这个时候企业就会进行补录，第二次大规模补录是在第二年的春招。

什么是意向书？

意向书：企业在对候选人（应聘人）的所有考核都通过之后，就会发放录用意向书，正式录用的邮件会在沟通薪资福利之后发送。接收到意向书不代表稳了，还有很多未知因素的存在（可能被毁），所以最好能拿3+个意向书比较稳。

什么是谈薪？

HR给收到录用意向书的候选人打电话沟通薪资福利，如果候选人接受，那么过后就会收到带薪资福利的正式录用offer，后续候选人就等着签约了。

有些公司没有谈薪的环节，直接发送带薪资福利的邮件。

如果手上有更好的offer，就可以跟HR谈条件，尝试提高薪资，也是我们平时所说的argue。听说现在一般要有阿里/字节的sp才能argue动battmdjkw这种级别的公司。一般argue成功的概率不大。

今年的情况是，阿里字节美团滴滴快手拼多多等公司都是在10月中下旬开始谈薪，京东小米携程顺丰等公司在国庆节之前就已经发送带薪邮件了。

简历制作与投递

简历制作

一份漂亮的简历，可以弥补其他方面的劣势，帮助你更大概率地通过简历筛选和获得面试机会。

  1. 必要信息一定要写：姓名，学校，毕业时间，年龄，邮箱，手机号码。这几个信息一定要写在简历的开头，方便HR和面试官的快速获得最重要的信息，并且HR后续将笔试/面试通知发送给你也是通过发邮件/打电话来进行的。

  2. 最好要写的信息：求职意向（如：后端开发工程师），当前所在城市等。求职意向方便HR快速判断你是否和相应岗位匹配，当前所在地可以对你进行远程视频/电话面试还是现场面试的选择，今年疫情影响，大部分公司都采取了无接触远程视频/电话面试了。

  3. 不要写有多余的信息：一份好的简历，不应该有多余的信息，比如技术岗最好不要写学生社团经历和CET4，通过CET6可以写上去，如果分比较高，可以标注一下分数。关于学生社团经历要不要写在简历上，技术岗最好不要写，非技术岗可以写，这一方面不同人有不同的观点，你可以投递几家公司对比试试就知道了。

  4. 重点：技术栈，项目经历一定要写，实习经历如果没有那就没办法了。面试的时候，主要是在二面和三面，面试对你的提问就是怼项目/实习。注意：这三个模块有些内容如果是重点/亮点的话，就加粗字体！

    强烈推荐 超级简历，在这个网站上面进行简历的制作，简单大方，不会花里胡哨！！

笔试/面试注意要点

笔试

技术岗的笔试内容组成：选择题+填空题+编程题。这三项都是可选项，不一定同时出现，但是大部分都是有2~3道编程题的，腾讯字节网易等公司只有编程题，而且不止3道！

笔试时间：一般90分钟，视情况而定。

笔试一般在牛客网或者另一个测评平台（到时候你就知道了）进行，有的公司会使用自家的网站。一般牛客网的在线测评系统比较友好，我还没遇到过卡顿的现象。

记得有一次，360笔试在某测评平台最后10分钟出现网站崩溃的现象，最后的解决方法是进行重新考试（个人自由选择是否重新考，取两个最高分计入总分）。

有些使用自家平台的，就是白板编程，没有代码智能提示，需要一个符号一个字母的打上去，而且不能评测你的代码是否能够通过测试用例，一般比较坑……

【注意】一般笔试的过程中，需要全程视频监控，牛客网还需要屏幕和*控，防止作弊！建议大家还是 **诚信考试为好！

面试

面试过程：面试开始会有自我介绍，最好控制在1分钟左右，姓名+学校+项目/实习经历即可，这也是给面试官看你简历的时间。面试时，表达要尽量做到逻辑清晰，思路完整。

面试结束：一般会有反问，这个时候可以向面试官询问公司内部和所在部门的工作氛围和技术栈等，或者对你的建议等，最好不要问面试结果。

一般面试的时候，多多少少都会有一点紧张，或者面试时突然某个知识点忘记了，这种情况是很正常的。平时，我们可以和共同作战的朋友进行模拟面试，互相问问题，加深记忆的同时还能训练思维和表达！

【注意】不要抢话， 注意礼貌，不要怼面试官，当然也有些面试官比较咄咄逼人，这个时候就看你自己怎么应对了。
